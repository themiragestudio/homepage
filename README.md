
### 管理命令
```bash
# 进入项目目录
cd "/Users/sysadmin/.openclaw/Mirage Studio/Projects/homepage/code/homepage-app"

# 开发模式（热重载）
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

```bash

npm install -g serve

# 部署生产版本
serve -s dist -l 8080
```
